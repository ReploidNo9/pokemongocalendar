# Contributing

Only publish event dates and times that can be traced to an official Pokémon GO source.

## Event rules

- Multi-day events should use all-day entries.
- Raid Hours, Spotlight Hours, Community Days, and Raid Days should use their actual time windows.
- Deadlines should be separate short reminder events.
- Never replace an existing UID when changing an event; stable UIDs allow subscribers to receive updates without duplicates.
- Do not publish rumors, leaks, or unverified raid counters.
